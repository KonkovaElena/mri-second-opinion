## Epistemic rule for external copy

Use GitHub-facing copy only if it stays within one of these boundaries:

1. implemented repository truth
2. clearly labeled target direction
3. research-informed context that is not presented as runtime proof

Never let short external summaries collapse those categories into one statement.
